# Fast-Flow Docs (Docusaurus)

Documentation is built with [Docusaurus](https://docusaurus.io/).

## Prerequisites

- **Node.js** ≥ 20
- **npm** (in project root: `npm install` for workspaces)

## Installation

In the project root:

```bash
npm install
```

Or only in the `docs/` folder:

```bash
cd docs
npm install
```

## Local development

```bash
# From project root
npm run docs:dev
```

or

```bash
cd docs
npm run start
```

Docs run at [http://localhost:3000](http://localhost:3000). Markdown changes reload automatically.

## Build

```bash
# From project root
npm run docs:build
```

or

```bash
cd docs
npm run build
```

Static files are in `docs/build/`. Local preview server: `npm run serve` (in `docs/`).

## Deployment (e.g. GitHub Pages)

```bash
cd docs
npm run deploy
```

For GitHub Pages with SSH: `USE_SSH=true npm run deploy`.  
Without SSH: `GIT_USER=<your-github-user> npm run deploy`.

## Mermaid diagrams

In Markdown, [Mermaid](https://mermaid.js.org/) diagrams are rendered via ` ```mermaid ` code blocks (e.g. in `docs/architektur.md`).

## Structure

- `docs/` – Markdown sources
- `src/` – React components, CSS
- `static/img/` – images, favicon, logo
- `sidebars.ts` – sidebar order (autogenerated from `docs/`)
