  WARNING: secrets.encryptionKey and/or secrets.jwtSecretKey are empty.
  The orchestrator will fail to start until these are set. Generate them with:

    python3 -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
    openssl rand -base64 32

  Then upgrade the release with:

    helm upgrade {{ .Release.Name }} fastflow \
      --set secrets.encryptionKey=<key> \
      --set secrets.jwtSecretKey=<key> \
      --reuse-values
